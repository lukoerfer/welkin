


                                    W E L K I N 



  What is this?
  -------------

  Welkin is an graphical RDF visualizer.
  


  Requirements
  ------------

  Welkin is a command line application or an applet and requires:
  
   o  A Java 1.5 or later compatible virtual machine for your operating system.



  Documentation
  -------------
  
  For more information on welkin, please look into the 'docs' folder or visit:
  
      http://simile.mit.edu/welkin/
